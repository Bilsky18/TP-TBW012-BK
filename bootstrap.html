Bootstrap

Bootstrap est un framework CSS open source créé par Twitter en 2011. Il permet de concevoir des interfaces web modernes et responsives (adaptées à tous les écrans) rapidement, sans repartir de zéro à chaque projet.

En tant que technicien systèmes et réseaux, maîtriser Bootstrap me permet de créer des interfaces d'administration réseau, des dashboards de supervision ou des pages de documentation claires et professionnelles.

Ce que je vais apprendre

Cette page évoluera au fil de la formation. Voici les notions qui seront abordées :

  • Intégrer Bootstrap dans un projet HTML
  • Utiliser le système de grille (grid) pour organiser le contenu
  • Utiliser les composants Bootstrap (boutons, cartes, navbar...)
  • Rendre une page responsive sans écrire de CSS
